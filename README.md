# Insurance Dataset Analysis

## AIML Assignment 16

This project contains Python programs for analyzing the Insurance Dataset using Pandas, Matplotlib, and Seaborn.

### Topics Covered
- Load the dataset
- Display first 10 rows
- Dataset information
- Missing value analysis
- Numerical and categorical columns
- Distribution plots
- Count plots
- Correlation heatmap
- Charges analysis
- Boxplots

### Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Dataset
- insurance.csv
